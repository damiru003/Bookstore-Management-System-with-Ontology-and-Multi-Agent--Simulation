## 🎯 KEY FEATURES

✨ Real-Time Ontology Visualization:
   • Shows all OWL individuals and their properties
   • Updates automatically every 20 simulation steps
   • Auto-refreshes when simulation completes

🔄 Interactive Controls:
   • Manual Refresh Button - Update ontology on demand
   • Export Button - Save ontology as .owl file
   • Statistics Bar - Shows entity counts

📊 Detailed Information Display:
   • Customer: ID, Name, Budget, Satisfaction, Purchases
   • Employee: ID, Name, Role, Managed Books
   • Book: ID, Title, Price, Genre, Stock (with warnings)
   • Transaction: ID, Customer, Book, Amount
   • Classifications: All SWRL-based categories

⚡ Performance Optimized:
   • Fast refresh (< 1 second)
   • Non-blocking updates
   • No impact on simulation speed

## 🚀 HOW TO USE

Step 1: Launch GUI
```powershell
D:/KDU/CSAT/MAS/.venv/Scripts/python.exe bookstore_gui.py
```

Step 2: Configure & Start Simulation
• Set parameters in "🎮 Setup & Control" tab
• Click "Start Simulation" button

Step 3: View Ontology
• Click on "🧠 Ontology" tab (4th tab)
• Explore the 5 sub-tabs:
  - 👥 Customers
  - 👔 Employees
  - 📚 Books
  - 💳 Transactions
  - 🏷️ Classifications

Step 4: Interact
• Click "🔄 Refresh Ontology" for manual update
• Click "💾 Export Ontology (.owl)" to save file
• View statistics bar for entity counts
